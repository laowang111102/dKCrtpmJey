# 热点新闻分析系统简介 python777

## 项目概述

本项目是一个基于Hadoop的热点新闻分析系统，旨在通过大数据分析技术，为用户提供实时的热点新闻动态及深入的新闻主题分析。

## 技术栈

- **后端：**
  - Python 3.7
  - Django
  - MySQL 5.7
- **前端：**
  - Vue.js
- **爬虫：**
  - Spider

## 功能模块

### 系统角色

- 管理员
- 用户
- 游客

### 管理员功能

管理员可以操作以下模块：

- 系统首页
- 个人中心
- 用户管理
- 新闻类型管理
- 主题标签管理
- 热点新闻管理
- 新闻评分管理
- 新闻资讯管理
- 系统管理

### 用户功能

用户可以操作以下模块：

- 系统首页
- 个人中心
- 新闻评分管理

### 游客功能

游客可以浏览以下内容：

- 首页
- 热点新闻
- 新闻资讯

## 系统运行环境

- 服务器：兼容Hadoop环境
- 客户端：现代浏览器

## 部署步骤

1. 确保服务器环境满足Hadoop运行条件。
2. 部署MySQL数据库。
3. 部署后端服务，配置Django与MySQL连接。
4. 部署前端Vue.js应用。
5. 爬虫部署，用于收集新闻数据。
